# Yelp Hybrid Recommender System

Sistema de recomendación híbrido desarrollado para el curso de Sistemas de Recomendación.

El sistema combina:

- Filtrado colaborativo mediante factorización matricial (SVD)
- Recomendación sensible al contexto
- Explicaciones de recomendaciones para el usuario

La aplicación permite descubrir negocios y restaurantes personalizados usando información de Yelp.

---

## Tecnologías utilizadas

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
